=====================================
  DCS Auto-GCI  -  Quick Start Guide
=====================================

INSTALL (Recommended)
---------------------
1. Run "DCS_AutoGCI_Setup.exe"
2. Follow the on-screen prompts
3. The installer will:
   - Install the app to your AppData folder
   - Place the required Lua scripts into your DCS Saved Games folders
   - Create Desktop and Start Menu shortcuts

INSTALL (Manual / Portable)
---------------------------
1. Copy "DCS Auto-GCI.exe" anywhere you like
2. Copy the Lua scripts to your DCS folders:
   - ThreatWarnerExport.lua  ->  Saved Games\DCS\Scripts\
   - ThreatWarnerHook.lua    ->  Saved Games\DCS\Scripts\Hooks\
   (If you use DCS Open Beta, also copy them to Saved Games\DCS.openbeta\)

RUN
---
1. Launch "DCS Auto-GCI" from the Desktop shortcut or Start Menu
   (or run the EXE directly if using the portable method)
2. Start DCS World and enter a mission
3. The app will automatically connect and begin announcing threats

UNINSTALL
---------
- Go to Windows Settings > Apps > Installed Apps
- Find "DCS Auto-GCI" and click Uninstall
  (or run the Setup.exe again and choose Remove)

MORE INFO
---------
Open "DCS_AutoGCI_README.html" in any web browser for full documentation,
including settings, voice configuration, OneCore voice setup, and more.
